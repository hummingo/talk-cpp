### build project
#### windows + visual studio 2019 **according to visual studio version config**
1. git clone https://github.com/hummingo/talk-cpp.git
2. mkdir build
3. cd build
4. cmake -G "Visual Studio 16 2019" -A x64 ..
5. cmake --build . --config Debug -v
6. open talk-cpp.sln

#### linux + gcc/g++
1. git clone https://github.com/hummingo/talk-cpp.git
2. mkdir build
3. cd build
4. cmake ..
5. make
